# KillBait Checker Extension

The KillBait Checker Extension helps you analyze news articles to detect clickbait and get AI-generated summaries in your preferred language. It is lightweight, easy to use, and designed for users of all technical levels.

If you see a value in the **API Key** field, the extension has already obtained it from your KillBait account.

### ğŸŒ What is the language selector for?
The language selector has three options:

- **Auto**: Uses your browser language (currently English or Spanish). Other languages default to English.  
- **Spanish**: Always uses Spanish.  
- **English**: Always uses English.  

The selected language determines which KillBait version the URL is checked against. You can submit articles in any language, but summaries will match your selected language.

### ğŸ§­ How do I use this extension?
1. Visit a news article in your browser.  
2. Click the extension button and select **Check Current URL**.  

- Already analyzed articles show the clickbait analysis, reason, voting buttons, and AI summary.  
- New articles appear as **In Process** in the history; check later for results.  

All actions are saved in **local history**, which you can review, reopen, compare, or clear anytime.  
The extension only runs on user demand; no background processes are active.
